Deploy real

Vercel para la app completa

El producto real necesita runtime Node.js para `app/api`, Redis, Stripe, Resend y streaming del chat. La opción correcta es desplegar `apps/web` en Vercel.

  1. Importar el repo en Vercel.
  2. Elegir `apps/web` como root.
  3. Cargar las variables de entorno.
  4. Deploy con datos mock o con proveedores reales.

Publish rápido

GitHub Pages para vidriera

Esta carpeta `docs/` se publica con GitHub Actions y sirve como demo comercial o portfolio técnico, sin depender del backend.

  1. Activar `GitHub Actions` en `Settings → Pages`.
  2. Hacer push a `main`.
  3. Compartir la URL pública de Pages.
